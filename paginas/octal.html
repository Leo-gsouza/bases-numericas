Sistema Octal

O sistema octal é um sistema de numeração em base 8, o que significa que utiliza oito dígitos diferentes para representar números: 0, 1, 2, 3, 4, 5, 6 e 7.

Octal para Decimal

A conversão de um número octal para decimal envolve multiplicar cada dígito do número octal pela potência de 8 correspondente à sua posição e, em seguida, somar os resultados. Exemplo:

Número octal 371

Identifique as potências de 8 para cada posição:

3 x 82 + 7 x 81 + 1 x 80

192 + 56 + 1 = 249

Portanto, o número octal 371 em decimal é 249.

Decimal para Octal

Para converter um número decimal para octal manualmente, você pode seguir um processo simples de divisão sucessiva por 8. Vou descrever os passos:

Exemplo de Conversão: 156 (Decimal) para Octal

156 ÷ 8 = 19 (quociente) e 4 (resto).

19 ÷ 8 = 2 (quociente) e 3 (resto)

2 ÷ 8 = 0 (quociente) e 2 (resto).

Agora, escreva os restos na ordem inversa: Número octal: 234.
